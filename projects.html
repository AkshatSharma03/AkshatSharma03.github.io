Agentic AI

Pravaha — Agentic Slack AI Assistant

RAG & agents
Python Django Celery Redis Claude API RAG
  • Architected an end-to-end agentic AI system embedded in Slack using Claude with native tool use; implemented an autonomous reasoning loop of up to 10 iterations so the agent can chain multiple tools per query.
  • Built a full RAG pipeline: embedded 221 document chunks with BAAI/bge-small-en-v1.5 (384 dimensions, CPU-only) and upserted them to Upstash Vector for semantic retrieval and grounded synthesis.
  • Engineered seven callable tools: web search, sandboxed Python execution, RAG knowledge base, live weather, Slack channel history, URL content parsing, and timezone-aware datetime.
  • Designed an asynchronous architecture with Celery and Redis so the Django view returns HTTP 200 in under 100ms while a worker processes the agentic loop; persisted per-thread memory in Django ORM.
  • Deployed a two-service architecture on Railway (web and Celery worker), formatting rich responses as Slack Block Kit messages with per-tool attribution.
